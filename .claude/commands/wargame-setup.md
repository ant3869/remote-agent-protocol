# Wargame Setup

Initialize a wargaming environment for project planning:
1. Create a `/tasks/` directory in the current workspace.
2. Create a `/war_games/` directory in the current workspace.
3. Create a `success.md` file with criteria: "A successful wargame must explore at least 2 layers of failures (second and third-order consequences). Must clearly state abort conditions."
4. Create a `ledger.md` file with the header `# Blockers and Required Inputs`.
5. Tell the user to add their task brief to `/tasks/` and then run `/wargame-simulate`.
